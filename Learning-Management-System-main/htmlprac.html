USING THE INTERNET CONNECTION WIZARD

To start the wizard for the first time, double-click The Internet icon on your desktop. If you need
to start the wizard again at a later time, click the Start menu, point to program, point to
Accessories, point to internet Tools, and then click Get on The Internet.

NOTE:    If you install this version of Internet Explorer and .then
            install an earlier version of Internet Explorer (for example,
            internet Explorer 2.0), the internet Connection wizard will 
            not work.

The wizard connects you to the Microsoft Internet Referral Service. This Service provides you
with a list of the service providers available in your area. To determine which service providers
are available, the wizard sends the following information to the Internet Referral Service.
